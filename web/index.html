如何建立並安全設定免費用量？
  1. 在 Google Cloud Console 建立專案並啟用帳單。
  2. 啟用 Maps JavaScript API 與 Places API (New)。
  3. 建立 API key，Application restriction 選 Websites，加入 https://xixa3333.github.io/Food_Lottery/*。
  4. API restrictions 只允許上述兩個 API，並在 Quotas 設定每日上限與帳單預算通知。
  5. Google 依 SKU 提供每月免費用量，額度可能調整；請查看官方定價頁。
